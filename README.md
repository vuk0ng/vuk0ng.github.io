# Netgen Systems - Infrastructure & Automation

Bienvenue sur le dépôt officiel de **Netgen Systems**. Ce projet regroupe la conception complète d'une infrastructure réseau critique et une solution logicielle d'automatisation.

## Le Projet
Netgen Systems est une étude de cas complète allant de la conception physique d'un bâtiment jusqu'à l'automatisation des configurations réseaux.

### Infrastructure & Ingénierie
* **Design de Bâtiment** : Plans de câblage structuré et optimisation des salles serveurs.
* **Datacenter** : Implémentation de solutions de refroidissement (Cooling) et haute disponibilité.
* **Segmentation Avancée** : Architecture LAN/WAN, adressage IPv4/IPv6 et routage dynamique (BGP).
* **Services IP** : Déploiement VoIP (IPBX), Vidéo sur IP, Wi-Fi Mesh et Active Directory.

### Netgen Config Tool
Au cœur de ce projet se trouve notre **Application Web d'automatisation**. 
Elle permet aux administrateurs de générer des scripts de configuration Cisco sans erreur humaine, garantissant une sécurité et une rapidité de déploiement accrues.

## Technologies Utilisées
* **Web** : HTML5, CSS3 (Modern Dark Design), JavaScript (ES6).
* **Réseaux** : Cisco IOS, Protocoles de routage, Firewalling.
* **Systèmes** : Windows Server (AD), IPBX, Monitoring.
